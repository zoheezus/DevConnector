# DevConnect
DevConnect is project through Traversy Media. It is meant to be a social media platform for developers.

## Tech Stack
Node.js <br />
Express <br />
React & Redux <br />
MongoDB

## Installation
You'll need to add your own keys file into the config folder and name is default.json <br />
```npm install``` <br/>
```cd client``` <br/>
```npm install``` <br/>
```cd ..``` <br/>

## Get App Running
```npm run dev```
